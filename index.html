Tutorial de um pro

Sobre:

Muitas pessoas reclamam que o jogo é muito difícil solo, mas não é tão complicado quanto muitos pensam.
Elas reclamam pois não levam o loadout certo para as missões, o que pode ser prejudicial para sua run.
Mas ao contrário do que muitos pensam, você apenas precisa de uma pistola para completar a missão, o mais importante do que o seu loadout ou gadgets são suas skills.

Loadout mais benéfico para solo:

  1. Taser: Você pode usar um taser para cuidar da maioria dos inimigos
  2. Polarizer: Você pode usar isso para silenciar seus passos e poder correr mais rápido
  3. Escudo: O escudo é um dos itens mais essenciais para se defender de tiros inimigos
  4. LockPick: Você pode usar esse item para abrir portas sem alertar seus inimigos
  5. Flashbangs: Você pode usar as flashbangs para jogar pela porta para nocautear seus inimigos e não correr nenhum risco
  6. Escudo: Você pode usá-la para checar se inimigos estão na sala que você precisa entrar

Skills:

A skill tree será sua melhor amiga para esse jogo. A skill tree de negociador é a melhor em termos de usabilidade, pois você pode fazer os inimigos de refém pela parede, o que deixa o jogo mais fácil.
